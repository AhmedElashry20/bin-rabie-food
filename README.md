# Bin Rabie Food | بن رابعي فود

Corporate website for **Bin Rabie Food**, a Saudi company specializing in logistics services for the restaurant sector, providing ready fleets including cars and motorcycles.

![Hero](screenshot.png)

## Features

- Modern dark theme with gradient design
- Animated statistics counters
- Services showcase section
- Partners carousel
- WhatsApp integration
- Fully responsive design

## Tech Stack

- HTML5 / CSS3 / JavaScript
- Custom animations & transitions
